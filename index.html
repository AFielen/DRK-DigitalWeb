NIS-2 Audit & NIS2 Manager — Unser Flaggschiff

Das umfassendste Open-Source-Toolset zur NIS-2-Betroffenheitsanalyse für DRK-Verbände. Der NIS-2 Audit liefert Self-Check, Kostenrechner, Management-Briefing und Lieferketten-Prüfung — alles Zero-Data. Der NIS2 Manager ergänzt den Audit um kontinuierliche NIS2-Dokumentation und laufende Compliance-Verwaltung.

Unsere Haltung

Digital ohne Kompromisse beim Datenschutz — die meisten Daten verbleiben auf dem Gerät.

Zero-Data & DSGVO

Fast alle Tools arbeiten ohne Server und Datenbank. Einige wenige (aktuell DRK Visitenkarte, künftig DRK Teamkennwort und DRK AutoDMS) nutzen Datenbanken — selbstverständlich DSGVO-konform auf Hetzner-Servern in Deutschland.

Open Source

Alle Tools quelloffen auf GitHub. Jeder Verband kann hosten und anpassen.

Sofort nutzbar

Keine Installation, keine Registrierung. Browser öffnen und loslegen.

Konsequent datenschutzkonform

Keine Cookies, keine externen Tracking-Dienste, keine US-Cloud-Anbieter. Alle Tools laufen nach dem DRK Goldstandard — Hosting bei Hetzner (Deutschland), Fonts lokal, keine US-Cloud, keine Tracker, keine externen CDN. Die meisten Tools speichern Daten nur im Browser, wenige Apps nutzen DSGVO-konforme Datenbanken.
