Unsere Haltung

Digital ohne Kompromisse beim Datenschutz — die meisten Daten verbleiben auf dem Gerät.

Zero-Data & DSGVO

Fast alle Tools arbeiten ohne Server und Datenbank. Einige wenige (DRK Teamkennwort, DRK AutoDMS, DRK Visitenkarte) nutzen Datenbanken — selbstverständlich DSGVO-konform.

Open Source

Alle Tools quelloffen auf GitHub. Jeder Verband kann hosten und anpassen.

Sofort nutzbar

Keine Installation, keine Registrierung. Browser öffnen und loslegen.

DSGVO-konform by Design

Keine Cookies, keine externen Tracking-Dienste, keine US-Cloud-Anbieter. Alle Tools laufen nach dem DRK Goldstandard — Hosting bei Hetzner (Deutschland), Fonts lokal. Die meisten Tools speichern Daten nur im Browser. Wenige Apps nutzen DSGVO-konforme Datenbanken.
